# Айна Гард — лендинг

Статический сайт: `index.html`, стили в `css/`, скрипты в `js/`.

## Ссылка на сайт (GitHub Pages)

После включения Pages в настройках репозитория сайт будет доступен по адресу:

**https://qspehu-sketch.github.io/aina-gard-site/**

В GitHub: **Settings → Pages → Build and deployment → Source: Deploy from a branch → Branch: `main` → folder `/ (root)` → Save.**

Первый деплой может занять 1–2 минуты.

## Локальный просмотр

Откройте файл `index.html` в браузере или запустите простой сервер в этой папке.
