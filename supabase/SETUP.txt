=== Админка ainai.pro + Supabase — что сделать вам (один раз) ===

1) Зайдите на https://supabase.com → Sign up → New project.
   Запомните пароль к базе (он нужен только самому Supabase).

2) Когда проект создался: слева «SQL» → New query.
   Откройте файл supabase/schema.sql из этого репозитория, скопируйте ВЕСЬ текст,
   вставьте в редактор и нажмите Run (или Ctrl+Enter).
   Должно быть без ошибок внизу.

3) Ключи для сайта:
   Project Settings (шестерёнка) → API.
   • Project URL — вставьте в js/supabase-config.js в SUPABASE_URL
   • anon public — вставьте в SUPABASE_ANON_KEY
   Сохраните файл и залейте на GitHub (или опубликуйте как обычно).

4) Создайте себе пользователя для входа в админку:
   Authentication → Users → Add user → укажите email и пароль.
   (Если включено «подтверждение почты», зайдите в Authentication → Providers → Email
   и временно отключите «Confirm email», чтобы не ловить письмо при первом входе.)

5) Откройте в браузере: https://ainai.pro/admin.html
   Войдите email/паролем из шага 4.

6) Главная страница:
   После правок карточек в админке обновите главную (F5). Статьи и отзывы появятся
   на сайте только если у записи включено «Опубликовано» / «Показывать на сайте».

Полезные адреса:
• Админка: /admin.html
• Форма заявки: внизу главной страницы

Если что-то «не сохраняется» — откройте консоль браузера (F12 → Console) и напишите,
какая там красная ошибка (можно скрин).
